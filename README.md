# 宴会注文集約システム

モバイルオーダーに対応していない飲食店で、
参加者が注文を登録し、幹事が注文を集約・管理できるWebアプリケーションです。

## 作成背景

大人数の飲み会では、幹事が口頭で注文を取りまとめる負担が大きく、
注文漏れや重複注文が発生しやすいという課題があります。

この課題を解決するため、本システムを開発しました。

## 主な機能

- 宴会作成
- 参加コードによる参加
- 注文登録
- 商品別集計
- 注文差し戻し
- 参加者メモ
- 過去の宴会設定複製
- 会計補助

## 使用技術

- PHP 8.x
- Laravel 12
- MySQL
- HTML
- CSS
- JavaScript
- Git / GitHub
